Web дизайн – це те по чому “зустрічають” | web дизайн и веб сайт Кожна web студія починає створювати веб сайт з розробки і вибору web дизайну. Для більшості, web дизайн це і початок, і найголовніша частина, тому що більшість сайтів, таких як сайти-візитки, корпоративні сайти, мають іміджеву, інформативну ціль. Тобто, для таких сайтів головне гарно виглядати, справити гідне враження. Коли ми говоримо про веб дизайн, доречно згадати і приказку «зустрічають по одежі, проводжають по уму». Так і є!
